COMMAND
======================

### 1.1. 삭제
##### 1.1.1. 폴더 삭제
```
r : 파일 디렉토리 함께 삭제하기
f : 파일 유무와 상관없이 삭제하기
rm -rf '삭제할 폴더명'
```

### 1.2. 용량
##### 1.2.1. 디렉토리 용량
```
du (Disk Usage)
-c --total 총량 표시
-m MB
-s --summarize 총량만 표시
-h 읽기 좋은 용량 단위로 표시
| sort -n 용량 오름차순 (-h 옵션 대신 -b,-k,-m과 함꼐 사용)
| sort -rn 용량 내림차순 (-h 옵션 대신 -b,-k,-m과 함꼐 사용)

ex)
du -sh /usr/local/
```
